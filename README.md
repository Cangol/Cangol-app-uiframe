### Usage
-
Maven

     <dependency>
         <groupId>mobi.cangol.mobile</groupId>
         <artifactId>uiframe</artifactId>
         <version>1.0.58</version>
         <type>aar</type>
     </dependency>
Gradle
 
    compile 'mobi.cangol.mobile:uiframe:1.0.58@aar'
